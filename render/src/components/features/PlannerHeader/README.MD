# Planner Header

Description: Header section displaying time, timeline, pomodoro timer, and status symbols.

⚡ Composable component with configurable widget layout.

## Structure

```
PlannerHeader/
├── index.tsx               # Main component with configurable layout
├── README.MD               # This file
└── widgets/                # Reusable widget components
    ├── TimeWidget.tsx           # Current time display
    ├── TimelineWidget.tsx       # Visual timeline/progress indicator
    ├── PomodoroWidget.tsx       # Pomodoro timer widget
    └── SymbolBlock.tsx          # Status symbols/icons
```
