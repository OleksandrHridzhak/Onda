# Contributing to Project

Thanks for your interest in contributing! Please follow these simple guidelines:

1. **Fork and clone** the repository.
2. **Create a branch** for your changes: `git checkout -b feature/name-of-the-feature`.
3. **Write clear commits** and follow the existing code style.
4. **Run tests** before submitting a pull request.
5. **Open a pull request** describing your changes clearly.

Please be respectful and follow our [Code of Conduct](CODE_OF_CONDUCT.md).
